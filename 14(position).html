by deault sab static hote h agar hum kisi bhi image ko move karna chahte h to uske liye hume position property ka use karna padega position property ke 4 values hote h 1. static 2. relative 3. absolute 4. fixed 5. sticky static->this is the default value of position property relative->relative position is used to move an element relative to its normal position.ismain top,left,right,bottom property ka use hota h and ik kisi ko move kiya h toh uska space wahi rehta h absolute->absolute position is used to move an element relative to its parent element.ismain top,left,right,bottom property ka use hota h and agar kisi ko move kiya h toh uska space waha se chala jata h.overlapping hoti h fixed->fixed position is used to move an element relative to the browser window or the viewport .it does not move when the window is scrolled sticky->sticky position is used to move an element relative to its parent element until it reaches a certain point and then it stays fixed.we have to use top,left,right,bottom property to set the point where the element should stick
