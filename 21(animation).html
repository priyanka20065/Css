The animation-direction property specifies whether an animation should be played forwards, backwards or in alternate cycles. The animation-direction property can have the following values: normal - The animation is played as normal (forwards). This is default reverse - The animation is played in reverse direction (backwards) alternate - The animation is played forwards first, then backwards alternate-reverse - The animation is played backwards first, then forwards The animation-timing-function property specifies the speed curve of the animation. The animation-timing-function property can have the following values: ease - Specifies an animation with a slow start, then fast, then end slowly (this is default) linear - Specifies an animation with the same speed from start to end ease-in - Specifies an animation with a slow start ease-out - Specifies an animation with a slow end ease-in-out - Specifies an animation with a slow start and end cubic-bezier(n,n,n,n) - Lets you define your own values in a cubic-bezier function fill-mode property is used to specify the style of the element before and after the animation. It can have the following values: none - Default value. The element will not retain the style from the animation forwards - The element will retain the style from the last keyframe backwards - The element will retain the style from the first keyframe both - The element will retain the style from the first keyframe and the last keyframe short hand property for animation is: animation: name duration timing-function delay iteration-count direction fill-mode play-state;
